-----: Algorithm for Travelling Salesman :----

C ({1}, 1) = 0
for s = 2 to n do
    for all subsets S Є {1, 2, 3, … , n} of size s and containing 1
        C (S, 1) = ∞
    for all j Є S and j ≠ 1
        C (S, j) = min {C (S – {j}, i) + d(i, j) for i Є S and i ≠ j}
Return minj C ({1, 2, 3, …, n}, j) + d(j, i)
